# 环境变量设置 - 快速参考

## 🎯 在哪里设置？

**GitHub 仓库 → Settings → Secrets and variables → Actions → Repository secrets**

## 📝 需要添加的 3 个变量

| 变量名称 | 说明 | 示例值 |
|---------|------|--------|
| `NEXT_PUBLIC_APP_ID` | 应用 ID | `app-xxxxxxxxxxxxx` |
| `NEXT_PUBLIC_APP_KEY` | API Key | `app-xxxxxxxxxxxxxxxxxxxxxxxxxxxxx` |
| `NEXT_PUBLIC_API_URL` | 外部 API URL | `https://api.dify.ai/v1` |

## 🚀 5 步完成设置

### 步骤 1️⃣
打开 GitHub 仓库，点击顶部的 **Settings**

### 步骤 2️⃣
左侧边栏找到 **Secrets and variables** → 点击 **Actions**

### 步骤 3️⃣
点击右上角的 **New repository secret** 按钮

### 步骤 4️⃣
填写表单：
- **Name**: 输入变量名称（如 `NEXT_PUBLIC_APP_ID`）
- **Secret**: 输入对应的值

### 步骤 5️⃣
点击 **Add secret** 保存，然后重复步骤 3-5 添加其他变量

## ✅ 完成后应该看到

```
Repository secrets (3)
  ✅ NEXT_PUBLIC_APP_ID      Updated ...
  ✅ NEXT_PUBLIC_APP_KEY     Updated ...
  ✅ NEXT_PUBLIC_API_URL     Updated ...
```

## ⚠️ 注意事项

1. **变量名称必须完全一致**（区分大小写）
2. **API URL 格式**：`https://api.dify.ai/v1`（包含 `/v1`）
3. **设置后无法再查看值**（只能更新或删除）
4. **需要管理员权限**才能添加 Secrets

如果遇到问题，请检查：

- API URL 格式是否正确
